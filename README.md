# SDL pong
![Pong](/images/pong.png "Pong")

## How to play 

1. Press <kbd>Space</kbd> to start a new game.
2. <kbd>W</kbd> and <kbd>S</kbd> control the left bar
3. <kbd>Up</kbd> and <kbd>Down</kbd> control the right bar
